# 🚀 AI Website Copy Generator for Local Businesses

## 📌 Project Overview
This project is an AI-powered website copy generator designed to create high-converting content for local businesses like cafes, salons, gyms, and clinics.

It uses structured prompts with AI tools (ChatGPT) to generate professional website content quickly and efficiently.

---

## 🎯 Objective
To build a system that generates:
- Homepage content
- Service descriptions
- Call-To-Action (CTA) sections

using prompt engineering techniques.

---

## 🛠️ Tools Used
- ChatGPT
- Google Docs / MS Word
- GitHub

---

## 💡 Features
- Generates engaging homepage copy with clear value proposition
- Creates service descriptions tailored to business type
- Produces strong and persuasive CTA sections
- Supports tone customization (casual, modern, premium)
- Simple and beginner-friendly workflow
